# Parameter Reference

This file records the initial low-gain commissioning parameters used by
`ur5e_direct_torque_anisotropic_impedance_tests.script`. Treat these values as
free-space validation settings, not production lamination-process settings.

## Shared Compensation Scales

All tests currently use the same direct-torque friction compensation scale
vectors:

| Parameter | Value |
| --- | --- |
| `viscous_scale` | `[0.9, 0.9, 0.8, 0.9, 0.9, 0.9]` |
| `coulomb_scale` | `[0.8, 0.8, 0.7, 0.8, 0.8, 0.8]` |

These are passed into `direct_torque()` and are separate from the extra torque
commands generated by the test controllers.

## Test 0: Zero Extra Torque

Purpose: enter `direct_torque()` with zero extra commanded torque.

| Parameter | Units | Value |
| --- | --- | --- |
| Duration limit | s | `1.0` |
| `dtau_lim` | Nm/control step | `[0.04, 0.04, 0.03, 0.008, 0.008, 0.005]` |
| `q_err_lim` | rad | `[0.03, 0.03, 0.03, 0.04, 0.04, 0.05]` |

This test has no stiffness. The joint drift guard is intentionally tight.

## Test 1: Joint Damping Only

Purpose: validate low joint damping against measured joint velocity.

| Parameter | Units | Value |
| --- | --- | --- |
| Duration limit | s | `3.0` |
| `Dq` | Nm/(rad/s) | `[0.25, 0.25, 0.18, 0.020, 0.020, 0.012]` |
| `tau_lim` | Nm | `[3.0, 3.0, 2.5, 0.60, 0.60, 0.40]` |
| `dtau_lim` | Nm/control step | `[0.035, 0.035, 0.025, 0.006, 0.006, 0.004]` |
| `q_err_lim` | rad | `[0.08, 0.08, 0.08, 0.10, 0.10, 0.12]` |

Control law:

```text
tau = -Dq * qdot
```

## Test 2: Joint-Space Anisotropic Impedance Hold

Purpose: validate a soft joint-space hold around the starting joint pose.

| Parameter | Units | Value |
| --- | --- | --- |
| Duration limit | s | `5.0` |
| `Kq` | Nm/rad | `[2.0, 2.0, 1.5, 0.15, 0.15, 0.08]` |
| `Dq` | Nm/(rad/s) | `[0.35, 0.35, 0.25, 0.030, 0.030, 0.018]` |
| `tau_lim` | Nm | `[4.0, 4.0, 3.0, 0.80, 0.80, 0.50]` |
| `dtau_lim` | Nm/control step | `[0.040, 0.040, 0.030, 0.008, 0.008, 0.005]` |
| `q_err_lim` | rad | `[0.12, 0.12, 0.12, 0.16, 0.16, 0.20]` |
| `q_err_clip` | rad | `[0.08, 0.08, 0.08, 0.10, 0.10, 0.12]` |

Control law:

```text
tau = Kq * (q_ref - q) - Dq * qdot
```

## Tests 3 and 4: Base-Frame Cartesian Translational Impedance

Purpose: validate a base-frame translational impedance controller around the
starting TCP pose. Test 3 uses no bias. Test 4 shifts the equilibrium by
0.003 m in base-frame +X and is gated by `ALLOW_TEST_4_BIASED_MOTION`.

| Parameter | Units | Value |
| --- | --- | --- |
| Test 3 duration limit | s | `5.0` |
| Test 4 duration limit | s | `4.0` |
| Test 3 bias | m | `[0.0, 0.0, 0.0]` |
| Test 4 bias | m | `[0.003, 0.0, 0.0]` |
| `Kx`, `Ky`, `Kz` | N/m | `100.0`, `60.0`, `20.0` |
| `Dx`, `Dy`, `Dz` | N/(m/s) | `20.0`, `15.0`, `10.0` |
| `Dq_null` | Nm/(rad/s) | `[0.08, 0.08, 0.06, 0.010, 0.010, 0.006]` |
| `tau_lim` | Nm | `[4.0, 4.0, 3.0, 0.80, 0.80, 0.50]` |
| `dtau_lim` | Nm/control step | `[0.040, 0.040, 0.030, 0.008, 0.008, 0.005]` |
| Cartesian error abort | m | `0.060` on X, Y, and Z |
| Cartesian error clip | m | `0.025` on X, Y, and Z |
| `q_err_lim` | rad | `[0.16, 0.16, 0.16, 0.20, 0.20, 0.25]` |

Control law:

```text
F = K * (x_ref - x) - D * xdot
tau = J^T * [Fx, Fy, Fz, 0, 0, 0] - Dq_null * qdot
```

Only translation is controlled. Orientation impedance and surface-frame
impedance are intentionally omitted from these commissioning tests.

## Tuning Notes

- Change only one parameter group at a time.
- Re-run the static validator after edits.
- Re-run all earlier tests that could be affected by a shared parameter change.
- Preserve the test 4 opt-in gate for any intentional Cartesian bias test.
- Do not tune these free-space values in contact with a surface.
